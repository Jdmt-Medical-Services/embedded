# jdmt

## Requirement

- sam & aws cli.
- .env File with the following entries:
    - accessKeyId
    - secretAccessKey
    - region
    - gkey --> Google-Maps-API-Key
    - endpoint
    - query url
    - data url

 ## Deployment

Upload src and lambda directory to jdmt.develop.

```npm run upload```

## Local testing

starts the frontend and backend (with hot reloading) locally.

```npm start```
