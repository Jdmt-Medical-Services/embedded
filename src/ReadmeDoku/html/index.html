Introduction

This documentation will you give a short overview of the code that is used on the embedded device for the JDMT data logger project. It starts with a manual which describes how to set up the configuration of the device, followed by a description of the headers and functions used in the programm. For coding Visua Studio Code with the PlatformIO plug in was used, if you're using a diffrent enviroment some descriptions may differ.

Device configuration

The whole configuration of a device is set up in the config.h file. At the top of this header file you can configure the mode in which the emedded device should run and some settings over the #define precompile settings.
